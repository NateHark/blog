Flexbox 101

Why do we need another layout mode?

CSS Layout Modes

Well known and widely supported

  • absolute (aka positioned)
  • block
  • table

Up and coming

  • flexbox

Don't hold your breath

  • grid

The Challenge

Build a responsive "Edit Profile " form with the following behaviors:

  • Horizonal label/input layout
  • Labels are truncated with ellipsis when needed
  • Vertical layout at a width of 600px or less

First, some necessary background information

Flexbox Terminology

https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout/Using_CSS_flexible_boxes

flex-direction

Defines the direction of the main axis

flex-direction: row;

flex items are laid out in a row

.flex-container {
  display: flex;
}

.flex-item {
  flex: 1 1 auto 
}

  
1

  
2

flex-direction: column;

flex items are laid out in a column (stacked)

.flex-container {
  display: flex;
  flex-direction: column;
}

.flex-item {
  flex: 1 1 auto;
}

  
1

  
2

flex-direction: row-reverse;

the same as row except the order of the flex items is reversed

.flex-container {
  display: flex;
  flex-direction: row-reverse;
}

.flex-item {
  flex: 1 1 auto;
}

flex-direction: column-reverse;

the same as column except the order of the flex items is reversed

.flex-container {
  display: flex;
  flex-direction: column-reverse;
}

.flex-item {
  flex: 1 1 auto;
  height: 50px;
}

flex-wrap

Determines whether flex items can wrap to multiple lines.

Allows you to control the direction in which items are wrapped

flex-wrap: wrap;

Allows wrapping of flex items onto multiple lines

.flex-container {
  display: flex;
  flex-wrap: wrap;
}

.flex-item {
  flex: 1 1 400px;
  height: 50px;
}

flex-grow

<number> - Specifies the grow factor

flex-shrink

<number> - Specifies the shrink factor

flex-basis

<number/percent/calc()> - the initial main size of the flex item

flex: <grow> <shrink> <basis>

shorthand for setting all three properties

.flex-container {
  display: flex;
}

.flex-item {
  flex: 1 1 auto;
  height: 50px;
}

.flex-item:first-child {
  flex: 2 1 auto;
}

.flex-item:last-child {
  flex: 0 0 50px;
}

justify-content

determines how flex items are aligned on the main axis

justify-content: flex-start:

aligns flex items to the start of the flex container

.flex-container {
  display: flex;
  justify-content: flex-start;
}

.flex-item {
  flex: 0 0 auto;
  height: 100px;
  width: 100px;
}

  
1

justify-content: flex-end;

aligns flex items to the end of the flex container

.flex-container {
  display: flex;
  justify-content: flex-end;
}

.flex-item {
  flex: 0 0 auto;
  height: 100px;
  width: 100px;
}

  width: 100px;

align-items

defines how items are aligned along the cross axis

align-items: stretch;

(default) items are stretched. Limited to width/height specified for each item

  height: 400px;
}

.flex-item {

Horizontal + Vertical Centering

Without display: table; or transform()

.flex-container {
  display: flex;
  height: 500px;
  justify-content: center;
  align-items: center;
}

.flex-item {
  flex: 0 1 auto;
}

  
I'm centered in both directions. Neat!
